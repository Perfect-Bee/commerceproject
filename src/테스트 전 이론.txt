Step02 : 객체 지향 설계를 적용해 순서 제어를 클래스로 관리하기

`CommerceSystem` 클래스 생성하기
    >>> src/utils/CommerceSystem.java
    - 커머스 플랫폼의 상품을 관리하고 사용자 입력을 처리하는 클래스입니다.
    - `Product`를 관리하는 리스트가 필드로 존재합니다.
        >>> main에 있는거 옮기기
        >>> class CommerceSystem {
        >>>    List<Product> products;
        >>> }
    - `main` 함수에서 관리하던 입력과 반복문 로직은 이제 `start` 함수를 만들어 관리합니다.
        >>> commerceSystem.start();으로 실행하도록 변경

    - `List<Product> products` 는 `CommerceSystem` 클래스 생성자를 통해 값을 할당합니다.
        >>> CommerceSystem system = new CommerceSystem();
        -  `CommerceSystem` 객체를 생성하고 사용하는 `main` 함수에서 객체를 생성할 때 값을 넘겨줍니다.
