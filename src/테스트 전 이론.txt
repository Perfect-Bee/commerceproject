도전 Lv.1 : 장바구니 및 주문하기 기능을 추가하기
1-1. 장바구니 생성 및 관리 기능 >>> 우선 장바구니 : ShoppingBag.java
-> 선택한 상품 추가
    >>> CommerceSystem.java에서 각각의 클래스에서 선택한(input) 상품(product)을 ShoppingBag에서 만든 리스트에 넣기
-> product 저장 및 조회 가능
-> 잘못된 선택 시 예외(이건 그냥 선택 시 예외처리)

1-2. 재고 관리 시스템
-> 장바구니에 넣을 때, 재고가 0인지 확인. 부족할 경우 알림
-> 주문 완료 시 재고 -1

1-3. 장바구니 출력 및 계산
-> 사용자가 주문을 시도하기 전에 장바구니에 담긴 모든 상품과 총 금액 : price 총합
-> 각 상품 이름, 가격, 수량 -> 총 금액 합계

1-4. 장바구니 담기 기능
-> 상품을 선택하면 담을 지 묻고, 추가와 취소 선택
-> 장바구니에 담은 거 출력(1-1의 조회 기능)

1-5. 주문 기능
-> 장바구니에 담긴 모든 항목 출력(1-1, 1-4 조회)
-> 합산 금액 총합 계산, 주문하기 선택 시 장바구니 초기화 + 해당 수량만큼 재고 빼기
